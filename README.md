# Docker Jekyll

> Docker-compose for Jekyll (Ruby)

Donwload using git

```sh
$ https://github.com/migueabellan/docker-jekyll.git
```

Build container

```sh
$ docker-compose build
```

Create and run new `jekyll` theme

```sh
$ docker-compose up
```

Or copy your jekyll theme in `web/` and run

```sh
$ docker-compose up
```

Access project

```sh
http://localhost:4000
```
